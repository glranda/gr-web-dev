Great Lakes 50th Anniversary Timeline

As a Senior Front-End Developer at Great Lakes (now Nelnet), I was tasked with building a single-page scrolling experience that conveys static information to users. Using jQuery, the Anniversary Timeline includes features like parallax scroll effects, timing-based events, and scroll-point based events.

What I did:

  • Styled the interface using HTML5 and LESS
  • Used JavaScript and jQuery to make the parallax effects, scroll-based events, and time-based events
  • Built the interface to be completely responsive and cross-browser functional
  • Collaborated with Designers to work through mockup issues, and worked with Product Owners to set deadline goals

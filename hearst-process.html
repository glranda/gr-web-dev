Hearst Newspapers

In 2019 I started as a Front-End Engineer at Hearst Newspapers in New York. My work was done in a PHP-based content management system, in which I delivered new features, added new websites, and performed maintenance on the Hearst network of over 40 individual sites.

What I did:

  • Redesigned existing websites using Sass mixins, LESS variables, and CSS3
  • Utilized JavaScript and jQuery to improve page load performance, used HTML5 element ‘srcset’ to lower image sizing
  • Maintained, improved, and added new features to the PHP based Content Management System
  • Led a project updating the ADA compliance of every website on the Hearst platform with semantic HTML5, aria-roles/labels, and JavaScript
  • Worked with 3rd party vendors to integrate new features into the codebase

Improving Page Load

Some of my favorite work at Hearst involved improving the page performance. I was able to help speed things up by responsively keeping image sizes down, preloading assets such as font files, and lazy loading content as the user scrolls.

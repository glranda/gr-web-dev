Attigo

As a Senior Front-End Developer at Nelnet, I was tasked with delivering a brand new site to a client via a CMS called Hubspot. The site features various re-usable templates, re-usable modules and functionality, and user controlled customization options that were put in place by me.

What I did:

  • Built a fully responsive, cross-browser compatible website in a CMS tool on a fast turnaround (one month)
  • Acted as a Product Owner and sort-of-UI Designer, as well as Developer, since a team was not provided to me
  • Used HubL (hubspot language), HTML5, and CSS3 to create templates, re-usable modules and plug-ins
  • Worked directly with Stake Holders to set deadline goals and run demo meetings
